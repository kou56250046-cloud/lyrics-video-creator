# 歌詞動画アプリ - 要件定義書

## プロジェクト概要

歌詞動画作成アプリは、音声ファイルと歌詞テキストを入力として、タイムスタンプ付きの歌詞動画を生成するWebアプリケーションです。ユーザーは歌詞の表示スタイル、アニメーション、背景などをカスタマイズし、最終的に動画ファイルとしてエクスポートできます。

### 主要機能
- 音声ファイルアップロードと再生
- 歌詞テキスト入力とタイムコード設定
- 歌詞表示スタイルのカスタマイズ（フォント、色、アニメーション）
- 背景画像/動画の追加
- ビジュアライザー機能
- 動画エクスポート（WebM形式）
- YouTubeアップロード機能

## 機能要件

### FR-001: 音声ファイル管理
- 音声ファイル（MP3, WAVなど）のアップロード機能
- ファイル情報の表示（ファイル名、サイズ）
- 音声再生コントロール（再生/一時停止、シークバー）
- 再生時間の表示

### FR-002: 歌詞管理
- 複数行の歌詞テキスト入力
- 各行に対するタイムコード設定
- タイムコードの手動設定と自動推定
- 歌詞リストの表示と編集

### FR-003: 歌詞スタイルカスタマイズ
- フォント選択（Noto Sans JP, Space Mono, Bebas Neue）
- 文字色とグラデーション設定
- 文字サイズ調整
- アニメーション効果（フェードイン/アウト、スケール、回転など）
- 文字ごとのアニメーション効果

### FR-004: 背景管理
- 背景色の設定
- 背景画像のアップロードと設定
- 背景動画のアップロードと設定
- 背景アニメーション（ズーム、パン、波形など）

### FR-005: ビジュアライザー
- 音声波形のリアルタイム表示
- ビジュアルエフェクト（パーティクル、グラデーション）
- ビジュアライザーのオン/オフ切り替え

### FR-006: 動画エクスポート
- キャンバスベースの動画生成
- WebM形式でのエクスポート
- 解像度選択（16:9, 9:16, 1:1）
- プログレス表示

### FR-007: YouTube連携
- YouTube APIを使用した動画アップロード
- 認証機能（Google OAuth）
- サムネイル画像設定
- アップロードプログレス表示

### FR-008: UI/UX
- ダークテーマのモダンなインターフェース
- レスポンシブデザイン
- ドラッグ&ドロップ操作
- リアルタイムプレビュー
- トースト通知

## 非機能要件

### NFR-001: パフォーマンス
- リアルタイム音声処理（Web Audio API）
- スムーズなアニメーション（60fps）
- 大容量ファイルの効率的な処理

### NFR-002: 互換性
- 主要ブラウザ対応（Chrome, Firefox, Safari, Edge）
- モバイルデバイス対応
- オフライン機能（PWA）

### NFR-003: セキュリティ
- ファイルアップロードのセキュリティチェック
- APIキーの安全な管理
- HTTPS通信

### NFR-004: 可用性
- 99.9%のアップタイム
- エラーハンドリングと回復機能
- データ永続化（localStorage）

## 技術要件

### TR-001: フロントエンド
- HTML5 Canvas API（動画生成）
- Web Audio API（音声処理）
- MediaRecorder API（動画録画）
- File API（ファイル処理）
- CSS3 Animation（アニメーション効果）

### TR-002: バックエンド
- Vercel Serverless Functions
- Groq Whisper API（文字起こし）
- YouTube Data API v3

### TR-003: ライブラリ
- formidable（ファイルアップロード）
- Web Audio API（音声分析）

## リスクと対策

### RA-001: パフォーマンスリスク
- 大容量ファイル処理時のメモリ不足
- 対策：ファイルサイズ制限、ストリーミング処理

### RA-002: API制限
- Groq APIのレート制限
- 対策：エラーハンドリング、キュー管理

### RA-003: ブラウザ互換性
- 古いブラウザでの機能制限
- 対策：ポリフィル使用、フォールバック機能
