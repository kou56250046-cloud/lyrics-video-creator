# アーキテクチャー設計

## システム概要

歌詞動画アプリは、クライアントサイド中心のWebアプリケーションで、音声ファイルから歌詞動画を生成します。バックエンドは最小限のAPIのみを使用し、主な処理はブラウザ上で行われます。

## コンポーネント構成

### 1. フロントエンド (index.html)

#### 主要コンポーネント
- **AudioManager**: 音声ファイルのアップロード、再生制御、Web Audio API管理
- **LyricsManager**: 歌詞テキストの解析、タイムコード管理
- **StyleManager**: 歌詞のフォント、色、アニメーション設定
- **BackgroundManager**: 背景画像/動画の管理とアニメーション
- **Visualizer**: リアルタイム音声波形表示とパーティクル効果
- **CanvasRenderer**: HTML5 Canvasを使用した動画生成
- **ExportManager**: MediaRecorder APIを使用した動画エクスポート
- **YouTubeUploader**: YouTube Data API v3を使用したアップロード

#### 状態管理
```javascript
// グローバル状態変数
var lyrics = [];           // 歌詞配列
var timecodes = [];        // タイムコード配列
var curIdx = -1;           // 現在の歌詞インデックス
var layers = [];           // 背景レイヤー
var particles = [];        // ビジュアライザーパーティクル
```

### 2. バックエンド (api/transcribe.js)

#### APIエンドポイント
- **POST /api/transcribe**: 音声ファイルを受け取り、Groq Whisper APIで文字起こしを実行

#### 処理フロー
1. formidableでファイルアップロード処理
2. ファイルをBlobに変換
3. Groq APIにFormDataで送信
4. タイムスタンプ付きの文字起こし結果を返却

### 3. 外部API連携

#### Groq Whisper API
- **用途**: 音声ファイルの文字起こしとタイムスタンプ生成
- **モデル**: whisper-large-v3
- **レスポンス形式**: verbose_json

#### YouTube Data API v3
- **用途**: 生成した動画のYouTubeアップロード
- **認証**: Google OAuth 2.0
- **機能**: 動画アップロード、サムネイル設定

## データフロー

### 動画生成フロー
```
音声ファイルアップロード
        ↓
    文字起こし (Groq API)
        ↓
タイムコード設定/調整
        ↓
スタイル設定 (フォント/色/アニメーション)
        ↓
背景設定 (画像/動画/アニメーション)
        ↓
Canvasレンダリング (60fps)
        ↓
MediaRecorderで動画キャプチャ
        ↓
WebMファイルエクスポート
```

### リアルタイム再生フロー
```
音声再生開始
        ↓
Web Audio APIで解析
        ↓
アナライザーで周波数データ取得
        ↓
ビジュアライザー描画
        ↓
歌詞同期 (タイムコードベース)
        ↓
Canvasアニメーション適用
```

## 技術仕様

### ブラウザAPI使用
- **Web Audio API**: 音声解析とビジュアライザー
- **HTML5 Canvas**: 動画レンダリング
- **MediaRecorder API**: 動画キャプチャ
- **File API**: ファイルアップロード
- **Fetch API**: API通信

### パフォーマンス最適化
- **RequestAnimationFrame**: スムーズなアニメーション
- **Canvasオフスクリーン**: 効率的なレンダリング
- **Web Workers**: 重い処理の分離（将来拡張）
- **メモリ管理**: 大容量ファイルのストリーミング処理

### セキュリティ対策
- **CORS設定**: API通信のセキュリティ
- **ファイル検証**: アップロードファイルのチェック
- **APIキー管理**: 環境変数での安全な管理
